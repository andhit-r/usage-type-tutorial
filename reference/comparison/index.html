    Rangkuman Konfigurasi yang Diperlukan¶

    Untuk menggunakan fitur ini dengan benar, pastikan konfigurasi berikut sudah ada:

    1. product.usage_type — buat usage type dengan kode unik dan account_id
    2. Produk — opsional: konfigurasi product.account untuk override akun per produk
    3. HR Expense (hr.expense_type):
    4. Set allowed_product_usage_ids → usage yang boleh dipilih
    5. Set default_product_usage_id → usage yang otomatis terisi
    6. Stock Move (stock.picking.type):
    7. Set debit_account_method + debit_usage_id (atau debit_account_code)
    8. Set credit_account_method + credit_usage_id (atau credit_account_code)
    9. Outsource Work (ir.model dari dokumen induk):
    10. Set outsource_work_usage_selection_method
    11. Set outsource_work_usage_ids (Fixed) atau outsource_work_usage_python_code (Python)
    12. Transaksi — pilih tipe/dokumen yang sudah dikonfigurasi, lalu input data
